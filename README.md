# Products App

Приложение на **React + TypeScript + Zustand** для работы со списком товаров.

## Демо

https://yas-7.github.io/products-app/

Для входа используйте:
```
email: emily.johnson@x.dummyjson.com
password: emilyspass
```

## Что реализовано

### Авторизация

- форма входа: почта / пароль
- валидация обязательных полей
- вывод ошибки API
- логика `Запомнить данные`
    - `localStorage`, если пользователь хочет сохранить сессию
    - `sessionStorage`, если сессия должна завершаться после закрытия вкладки/браузера

### Таблица товаров

- загрузка товаров из **DummyJSON Products API**
- серверная сортировка
- серверная пагинация
- поиск товаров через API
- refresh таблицы
- подсветка рейтинга красным, если он ниже `3.5`

### Добавление товара

- открытие модального окна по кнопке `Добавить`
- локальное добавление товара без сохранения в API
- toast-уведомление после успешного добавления

### Примечание по добавлению товара

На текущем этапе новый товар не отправляется в API и отображается локально на клиенте.
Это сделано специально, потому что по заданию сохранение через API пока не требуется.
